# Ghost Theme - primo

primo - Ghost Theme

## Setup

### Disqus

Add this code snippet to the code injection *Blog Header* area

```
<script>var disqus_shortname = "ghostprimo"</script>
```
Change the shortname according to your own blog

## Site Url

```
ghost config url https://yournewdomain.com
ghost restart
```
